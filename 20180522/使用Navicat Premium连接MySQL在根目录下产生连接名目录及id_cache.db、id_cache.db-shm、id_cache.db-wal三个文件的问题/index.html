使用Navicat Premium连接MySQL在根目录下产生连接名目录及id_cache.db、id_cache.db-shm、id_cache.db-wal三个文件的问题

2018-05-22

问题：

  使用Navicat Premium连接MySQL会产生id_cache.db、id_cache.db-shm、id_cache.db-wal三个文件在根目录下。

原因：

  新建连接时，未对连接的设置位置进行设置。

解决方法：

  在 设置位置 处 设置 设置位置目录。
  设置位置：选中连接->右键 编辑连接->选择 高级 选项->设置位置。
